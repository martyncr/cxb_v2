Board Cyber Governance Assessment

Select the maturity level (0–4) that best reflects your current position for each action. Levels are aligned to the CxB Cyber Governance Code of Practice and can be used as evidence of board oversight and continuous improvement.

Traffic light key

0–1 Minimal governance

>1–3 Progressive / mixed

>3–4 Good overall / leading

Admin mode

Use admin mode to prefill ratings and lock specific questions before sharing with the board (e.g. where minimum expected practice is set). Changes are stored in this browser only.

To lock an individual action, set a level then click the padlock icon next to it (while admin mode is on).
